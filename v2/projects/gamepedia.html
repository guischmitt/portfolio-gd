Gamepedia

Responsive Game Portal

GamePedia is a database of games, in style of IMDB, as well as a forum / Q&A repository. It features responsive design for desktop and mobile that takes in consideration the context and limitations of each platform.

The portal was designed with reponsiveness in mind from the very beginning.

Search results in desktop and mobile versions.

The article view takes full advantage of the larger screen state, while the mobile version uses gestures for its navigation.
